Combat System
Defines how character stats translate into combat values. These formulae apply to all seekers and creatures. Edit the coefficients and save to Firebase — the game reads them at combat start.
Derived Stats Formulae

Each formula: result = stat × coefficient + flat bonus per level. Stats: FOR=Fortitude, WRA=Wrath, INT=Intuition, EMP=Empathy, END=Endurance, MEM=Memory, CUN=Cunning, GNO=Gnosis.
